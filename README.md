# Flutter Expenses App

## Getting Started

The Expenses App, built using Flutter, provides a seamless way to manage
personal finances. Users can effortlessly add expenses, which are then
displayed on a dedicated page listing all recorded expenditures. The app
also features a comprehensive budget page, offering a clear overview of
spending across various categories. Each expense tag displays the amount
spent alongside a percentage, helping users track their financial habits.
This intuitive interface ensures that managing expenses is both simple and
efficient.
